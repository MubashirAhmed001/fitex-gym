# FITEX GYM WEBSITE

A simple luxury gym website made with Flask.

---

 What it does
- Home page with big hero image
- About, Membership, Contact pages
- Working contact form 

---

How to run it (super easy)

1. Download the project
   bash
   git clone https://github.com/MubashirAhmed001/fitex-gym.git
   cd fitex-gym

2. Install Flask
   bash
   pip install Flask
   
3. Run the site
   bash
   python app.py

4. Open in browser
   http://127.0.0.1:5000

Files
  app.py        - main code
  static/css/style.css - colors & style
  templates/   -  all HTML pages
